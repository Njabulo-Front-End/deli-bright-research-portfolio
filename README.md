# Deli-Bright Research Portfolio

A responsive personal academic portfolio website built for **Deli-Bright Nii Tettey Oku**, a chemistry researcher and PhD candidate at the University of South Africa (UNISA), specializing in computational chemistry, machine learning, and AI-driven drug discovery.

This project showcases academic achievements, research publications, presentations, honors, awards, and educational background in a professional and accessible web format.

---

## Overview

The portfolio is designed to present Deli-Bright’s scientific and academic journey in a clean, responsive, and interactive format.

It includes:

- Professional introduction and biography
- Academic publications
- Research presentations
- Honors and awards
- Educational background
- Contact and social links
- Achievements gallery

---

## Features

Optimized for:
